Fantasy Adventure Game

An immersive fantasy RPG built with Unreal Engine, featuring dynamic combat and exploration mechanics.

Project Concept

The goal of this project was to create an open-world RPG that balances combat, exploration, and storytelling. Players can interact with dynamic NPCs and uncover hidden secrets throughout the world.

Screenshots

Game Engine & Software Used

  • Unreal Engine 5
  • Maya (Character Animation)
  • Substance Painter (Texturing)
  • Blender 3D (Props & Environment)
  • AfterEffects (VFX Compositing)

Development Pipeline

The pipeline consisted of pre-production design, blockout levels in Unreal Engine, character rigging and animation in Maya, texturing in Substance Painter, lighting and rendering in Blender, VFX in AfterEffects, and final integration in Unreal Engine.
